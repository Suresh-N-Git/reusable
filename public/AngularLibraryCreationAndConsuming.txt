npx ng build reusable-table


It creates the library under `projects/reusable-table`.

Files to keep in `src/lib`:
- `reusable-table.component.html`
- `reusable-table.component.ts`
- `reusable-table.component.scss`
- `reusable-table.module.ts`
- `reusable-table.styles.scss`
- `table-export.service.ts`

`public-api.ts` exports the package surface:
```ts
export * from './lib/reusable-table.module';
export * from './lib/reusable-table.component';
```

Package metadata:
```json
{
  "name": "reusable-table",
  "version": "0.0.1",
  "peerDependencies": {
    "@angular/common": "^18.2.0",
```

Build and pack:
```bash
npx ng build reusable-table

cd dist/reusable-table
npm pack
```

Install in another project:
```bash
npm uninstall reusable-table
npm install reusable-table-0.0.1.tgz
```

Required dependencies:
```bash
npm install bootstrap
npm install xlsx jspdf jspdf-autotable file-saver
```

Import and styles:
```ts
import { ReusableTableModule } from 'reusable-table';
```
```scss
@import 'reusable-table/styles';
```

Component usage:
```ts
import { ReusableTableModule } from '../reusable-table/reusable-table.module';
import { ReUsableTableColumn, ReusableTableConfig } from '../reusable-table/reusable-table.component';
```

Example usage:
```html
<app-reusabletable
  [data]="pumpData"
  [columns]="reUsableTableColumn"
  [tableConfig]="reUsableTableConfig"
  (rowEdit)="onEdit($event)"
  (rowDelete)="onDelete($event)"
  (rowSelect)="onSelect($event)">
</app-reusabletable>
```
