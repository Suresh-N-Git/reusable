ReusableTable App Summary

One-page overview based only on repository evidence

What It Is

ReusableTable is an Angular 18 app plus a publishable library that demonstrates a configurable Material data table. The repo also includes a sample tableui page that wires demo data into the reusable component.

Who It's For

Primary Angular developers or teams that need a reusable, configurable table component for internal apps or libraries.

Not found in repo: a formal end-user persona or business audience.

What It Does

  • Displays tabular data with configurable columns, widths, alignment, and per-column types.
  • Supports sorting, pagination, and global search across rows.
  • Lets users show or hide columns from a toolbar menu.
  • Exports the current table view to CSV, Excel, or PDF.
  • Prints the table through a dedicated print flow.
  • Handles integer, number, currency, date, chip, multiline, link, and actions cells.
  • Emits row-level select, edit, and delete events for host apps.

How It Works

  • src/app/tables/tableui/tableui.component.ts builds sample data and column config, then passes both into app-reusabletable.
  • projects/reusable-table/src/lib/reusable-table.component.ts renders the table with MatTableDataSource, MatPaginator, and MatSort.
  • The component merges runtime tableConfig values with defaults, filters rows with a custom search predicate, and keeps the visible-column list in sync with the column menu.
  • projects/reusable-table/src/lib/table-export.service.ts handles CSV, Excel, PDF, and print output.
  • The library module imports Angular Material table, paginator, sort, form-field, input, icon, menu, checkbox, button, tooltip, and chips modules.
  • The root route redirects to /tableui.

How To Run

  • npm install
  • npm start
  • Open http://localhost:4200/ and use /tableui as the demo route.
  • Optional: npm run build and npm test.
